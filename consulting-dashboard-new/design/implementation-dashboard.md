## 🔧 技術的準備事項

### データベース
```bash
# 各サービスのPrismaスキーマをセットアップ
npm run db:generate:all  # 全サービスのクライアント生成
npm run db:push:all      # 全サービスのスキーマ反映
```

### 開発環境
```bash
# 複数サービスの同時開発
npm run dev:services     # 全サービスの監視モード
```

### テスト環境
```bash
# E2Eテストの準備
npm run test:e2e:setup   # Playwright環境構築
npm run test:integration # 統合テスト実行
```
