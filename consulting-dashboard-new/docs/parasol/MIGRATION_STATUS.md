# V2→V3 移行ステータス

**最終更新**: 2025-10-31
**GitHub Issue**: #188

## 📋 移行概要

パラソル設計フレームワークをV2.0（サービスベース）からV3.0（BCベース）の階層構造に移行します。

### 移行目標
- **V2.0構造**: `services/` → **V3.0構造**: `business-capabilities/`
- **階層関係の明確化**: BC層 ⊃ L3 Capability層 ⊃ Operation層
- **Issue #146整合性**: API WHAT/HOW分離構造の維持
