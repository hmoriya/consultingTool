## 📋 How: この操作の定義

### 操作の概要
メンバーのキャリア目標を明確化し、実現に向けた開発計画を支援する。計画的なキャリア開発により、メンバーの成長と組織の人材力強化を実現する。

### 実現する機能
- キャリア目標の設定と管理
- キャリアパスの提案
- 開発計画の作成と追跡
- メンタリングとコーチングの支援

### 入力
- メンバーのキャリア志向
- 現在のスキル・経験
- 組織のキャリアパス定義
- 開発機会情報

### 出力
- キャリア開発計画
- キャリアパス提案
- 開発活動の進捗
- メンタリング記録
