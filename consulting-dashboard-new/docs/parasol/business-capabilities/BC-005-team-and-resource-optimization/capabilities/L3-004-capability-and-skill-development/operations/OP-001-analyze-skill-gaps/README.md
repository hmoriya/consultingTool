## 📋 How: この操作の定義

### 操作の概要
現在のスキル状況と必要スキルのギャップを分析する。データドリブンなスキル分析により、戦略的なスキル開発計画を支援する。

### 実現する機能
- 現在のスキル保有状況の分析
- 必要スキルの定義と比較
- スキルギャップの定量化
- 優先度付けとギャップ分析レポート

### 入力
- 現在のスキルマトリックス
- 事業計画とプロジェクト計画
- 必要スキル定義
- 業界トレンドデータ

### 出力
- スキルギャップ分析レポート
- 不足スキルの優先順位
- スキル開発推奨事項
- 採用・育成計画への示唆
