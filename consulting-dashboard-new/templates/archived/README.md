## 🗄️ アーカイブポリシー

### アーカイブ対象となる条件

1. **v1.0仕様テンプレート**: パラソルドメイン連携セクションなし
2. **品質スコア60未満**: 実用性に問題があるテンプレート

4. **保守停止**: 更新・サポートが終了したテンプレート

### アーカイブ手順

1. **バックアップ作成**: 元ファイルの完全コピーを作成
2. **アーカイブ日付記録**: ファイル名に日付サフィックス追加
3. **代替テンプレート明記**: 推奨代替テンプレートの指定
4. **アクセス制限**: templates/ ディレクトリから削除
