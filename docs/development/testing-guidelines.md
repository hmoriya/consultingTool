# 動作確認

## 基本的な動作確認手順

1. **サーバーの再起動**
   - 起動しているサーバをKillして新たにサーバーを起動

2. **ブラウザでの動作確認**
   - ソースコード修正後は、Playwright MCPでログインと動作確認をする

3. **設計書との整合性確認**
   - ソースコード修正後は、設計書と相違がないか確認

## Playwright MCP使用ルール

### 絶対的な禁止事項

1. **いかなる形式のコード実行も禁止**
   - Python、JavaScript、Bash等でのブラウザ操作
   - MCPツールを調査するためのコード実行
   - subprocessやコマンド実行によるアプローチ

2. **利用可能なのはMCPツールの直接呼び出しのみ**
   - playwright:browser_navigate
   - playwright:browser_screenshot
   - 他のPlaywright MCPツール

3. **エラー時は即座に報告**
   - 回避策を探さない
   - 代替手段を実行しない
   - エラーメッセージをそのまま伝える
