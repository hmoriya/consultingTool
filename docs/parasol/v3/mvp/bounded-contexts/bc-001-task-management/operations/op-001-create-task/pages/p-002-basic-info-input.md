# P-002: 基本情報入力ページ

## 概要
タスクの基本情報（タイトル、説明、優先度等）を入力するフォームページ。

## 対応UseCase
- **UC-002**: 基本情報入力

## ページ構成
```
┌─────────────────────────────────────────────────────────────┐
│ 新規タスク作成 - ステップ2: 基本情報入力                        │
│                                                             │
│ 📝 タスクタイトル * (必須)                                   │
│ [________________________________] (5-100文字)            │
│                                                             │
│ 📄 タスク説明 * (必須)                                       │
│ [                                ] (20-2000文字)           │
│ [          テキストエリア          ]                         │
│ [                                ]                         │
│                                                             │
│ ⚡ 優先度 * (必須)                                          │
│ ○ 高 ○ 中 ● 低                                             │
│                                                             │
│ 🏷️ カテゴリ                                                │
│ [ドロップダウン選択▼]                                        │
│                                                             │
│ 🔖 タグ                                                     │
│ [タグ1] [タグ2] [+ 追加]                                    │
│                                                             │
│ [< 戻る] [下書き保存] [次へ >]                               │
└─────────────────────────────────────────────────────────────┘
```

## UI コンポーネント

### フォームバリデーション
```typescript
interface BasicInfoForm {
  title: string;      // 5-100文字
  description: string; // 20-2000文字
  priority: 'HIGH' | 'MEDIUM' | 'LOW';
  category: string;
  tags: string[];     // 最大10個
}
```
