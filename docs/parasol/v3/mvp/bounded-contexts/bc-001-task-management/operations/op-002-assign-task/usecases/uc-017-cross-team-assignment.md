# UC-017: クロスチーム アサインメント

## 概要
複数チーム間でのリソース共有とタスクアサインメントを最適化するユースケース。

## 基本フロー
```mermaid
graph TD
    A[開始] --> B[チーム間リソース確認]
    B --> C[クロスチーム候補検索]
    C --> D[チーム間調整]
    D --> E[アサイン実行]
    E --> F[完了]
```
